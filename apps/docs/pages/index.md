# Willkommen bei promantis Helpdesk

promantis Helpdesk ist eine vollständig quelloffene Helpdesk-Lösung, die Teams eine professionelle Alternative zu teuren Software-Produkten bietet. Unser Ziel: intuitive Software mit allen Funktionen professioneller Lösungen — einfach und benutzerfreundlich.

Diese Dokumentation behandelt die Einrichtung, den täglichen Einsatz und die Weiterentwicklung von promantis Helpdesk.
